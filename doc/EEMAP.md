# EEPROM MAP

```
 Addr     +----------------------------------------------+
 0x0000   | ee_status                           [1 byte] |
          | 0xDB (MAGIC) = initialized                   |
          | 0xFF (EE_EMPTY) = empty / erased             |
          +----------------------------------------------+
          | database[16]                     [768 bytes] |
          |                                              |
          |                                              |
 0x0001   | +-- User[0] ------------------- 48 bytes --+ |
          | | hash[32]   BLAKE2s(RFID)  0x0001..0x0020 | |
          | | name[16]   "...\0"        0x0021..0x0030 | |
          | +------------------------------------------+ |
 0x0031   | +-- User[1] ------------------- 48 bytes --+ |
          | | hash[32]                  0x0031..0x0050 | |
          | | name[16]                  0x0051..0x0060 | |
          | +------------------------------------------+ |
          |                     ...                      |
          |            (User[2] ... User[14])            |
          |                     ...                      |
 0x02D1   | +-- User[15] ------------------ 48 bytes --+ |
          | | hash[32]                  0x02D1..0x02F0 | |
          | | name[16]                  0x02F1..0x0300 | |
          | +------------------------------------------+ |
 0x0300   +----------------------------------------------+
```
